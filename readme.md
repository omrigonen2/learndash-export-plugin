# LearnDash Export Plugin

This plugin provides CSV exports for LearnDash data:

- **Courses** – exports course IDs and titles
- **Students** – exports user IDs, usernames and email addresses
- **Enrollments** – exports user IDs with the courses they are enrolled in

Upload the plugin to your WordPress site and activate it. You will find a new
"LearnDash Export" entry under **Tools** where you can choose the data types to
export.
